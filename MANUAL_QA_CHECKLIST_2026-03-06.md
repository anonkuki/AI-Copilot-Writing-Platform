# AI+ 人工联调清单

日期: 2026-03-06
适用版本: 当前工作区代码
目标: 逐项验证模型切换后，各 AI 功能在三种模型下都能正常运行，并确认自然语言助手不会误触发自动操作

## 一、测试前准备

1. 启动后端: `cd backend && npm run start:dev`
2. 启动前端: `cd frontend && npm run dev`
3. 确认已登录，且有一本文档完整的书，至少包含:
   - 世界观
   - 角色
   - 剧情线
   - 伏笔
   - 至少 1 章正文
4. 在 AI 面板顶部确认可选模型列表为:
   - `Pro/deepseek-ai/DeepSeek-V3.2`
   - `THUDM/GLM-4-9B-Chat`
   - `Pro/MiniMaxAI/MiniMax-M2.5`
5. 每切换一次模型，至少执行一次普通问答和一次自动操作类请求

## 二、每个模型都要执行的基础项

对每个模型重复以下步骤:

1. 切换到目标模型
2. 打开一本已有数据的书
3. 打开任意章节
4. 确认 AI 面板顶部显示当前模型名称正确
5. 记录开始测试的模型名、时间、书名、章节名

## 三、自然语言问答联调

目标: 验证普通自然语言输入只返回回答，不误触发自动创建/改写

### 1. 普通知识问答
输入:
`这个主角目前最大的动机是什么 请结合已有设定回答`

预期:
- 助手正常给出文字回答
- 不自动生成角色、剧情线、伏笔、创作计划
- 聊天消息下方不出现自动执行按钮，或 `suggestedActions` 为空

### 2. 解释类问答
输入:
`你觉得这本书目前的世界观优势和短板分别是什么`

预期:
- 返回分析性回答
- 不自动改正文
- 不触发全文分析或创意计划

### 3. 追问上下文
输入:
`为什么你刚才这么判断 请给我具体依据`

预期:
- 能承接上文继续解释
- 不要求固定关键词
- 不误触发 ACTIONS

### 4. 模糊自然语言请求
输入:
`我有点拿不准这一段是不是太平了 你先跟我聊聊看法`

预期:
- 返回讨论式建议
- 不直接自动改写正文
- 除非明确要求执行，否则不触发自动操作

## 四、自动执行类功能联调

### 1. 续写
输入:
`请继续写这一章`

预期:
- 聊天回复先说明续写方向
- 自动生成续写候选
- 出现“同意/不同意”之类确认操作
- 同意后内容插入当前章节

### 2. 改进/润色
输入:
`把这一章开头润色一下`

预期:
- 生成改写结果或 diff
- 前端显示可接受/拒绝的变更
- 接受后章节保存成功

### 3. 扩写
输入:
`把主角和反派第一次对话扩写得更紧张一些`

预期:
- 返回可应用的扩写结果
- 不出现空回复
- 变更可确认

### 4. 创意计划
输入:
`我想写一本黑暗修仙小说 先给我完整设定 主角团和前三章计划`

预期:
- 触发创意计划流式生成
- 返回结构化计划摘要
- 出现“确认执行计划”按钮

### 5. 执行创意计划
操作:
- 点击“确认执行计划”

预期:
- 世界观、角色、剧情线、伏笔、章节逐步创建
- 进度条或进度消息持续更新
- 结束后可在左侧数据面板看到新增内容

## 五、右侧工具面板联调

在同一章节正文下分别点击:

1. 校对
2. 拼字
3. 灵感
4. 妙笔

每项预期:
- 能流式返回结果
- 不报 4xx/5xx
- 文本内容与当前模型一致返回
- 切换模型后再次点击，仍能正常返回

## 六、内联润色联调

操作:
1. 选中一段 100 到 300 字正文
2. 触发内联润色

预期:
- 返回多条 suggestion
- 每条包含原文、建议替换、原因
- 不出现 `<<<FIND>>>` 等解析残留标签
- 接受/拒绝逻辑正常

## 七、辅助编辑联调

分别打开以下面板并点击 AI 辅助:

1. 角色编辑器
2. 世界观编辑器
3. 剧情线编辑器
4. 伏笔编辑器中相关补全入口

预期:
- 返回结构化字段建议
- JSON 解析成功
- 切换模型后仍能返回建议

## 八、全文分析联调

分别执行:

1. 伏笔分析
2. 角色弧线分析
3. 节奏分析
4. 全面分析

预期:
- 流式输出不为空
- 结束后有结构化建议
- 可从建议中触发后续创建/改进动作

## 十、异常项检查

如果出现以下问题，记录具体模型和输入:

1. 普通自然语言问答误触发自动执行
2. 返回空白回复
3. 返回模型不存在错误
4. 返回结构化 JSON 失败
5. 流式中断
6. 接受变更后正文未保存
7. 前端显示的模型与实际调用模型不一致

## 十一、建议的最小回归输入集

1. `这个角色现在的情绪状态是什么`
2. `请继续写这一段`
3. `帮我润色这一章开头`
4. `分析这本书目前的节奏问题`
5. `给我设计一个新的反派角色`
6. `我想写一本赛博修仙小说 给我前三章计划`
